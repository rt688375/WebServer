WebServer
=========

Para iniciar el servidor web solo es necesario compilarlo con el Makefile, después se escribirá "./ WebServer" en la terminar dentro de la carpeta donde se encuentre el archivo, éste usará el puerto 8080 por default y se podrá acceder a él escribiendo "localhost:8080/" en el navegador, seguido de la pagina que se desea buscar, si existe, el servidor la mandará al navegador, si no, mostrará un error 404 indicando que la pagina no fue encontrada
